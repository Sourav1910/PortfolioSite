Live Stock API

Live stock data is required while making any application related to Stock Market Analysis which is very essential for investing as well as trading purposes. To overcome this problem I have created this API which gives Live Stock Data completely free of cost. This API, was built using Cheerio library for web scraping in JavaScript and Node and Express were used for the endpoints.
